## Purpose

This page gives technically interested first-time readers a compact, STMS-specific visual map of the repository before they read the manuscript, figures, source materials, and claim-boundary documents.

It summarizes:

- the STMS-specific mission variable,
- thermal margin and mission-throughput preservation,
- radiative-load, storage/buffer, power-limit, sensing-delay, and actuator-authority channels,
- hybrid separated-barrier control interpretation,
- combined thermal-stress diagnosis,
- evidence hierarchy,
- repository reading order,
- explicit spacecraft / flight / product / safety non-claims.

All graph sections include replay controls so the reader can step through the visual logic interactively.
